# Vibe Lamp Core V1 PCB 约束

## 生产目标

- 用户不会焊接。
- 所有 V1 必需器件必须 SMT 工厂贴好。
- 不使用需要用户插针、焊线、飞线的设计。
- 板载 WS2812B 是 V1 主显示；外接 WS2812 扩展只是预留，不属于 V1 必需交付。

## PCB

- 尺寸：当前 Core V1 为 40 mm x 40 mm，圆角。
- 层数：2 层。
- 板厚：1.0 mm 或 1.2 mm。
- 阻焊颜色：建议黑色或白色，按外壳视觉选择。
- 表面处理：HASL/沉金均可；若预算允许，沉金更适合小焊盘和外观。

## 布局

正面：

- WS2812B 状态灯放中心。
- USB-C 放一边，外壳开口要预留插拔空间。
- 清网按钮放边缘，外壳可做针孔或小按键柱。

背面或非视觉区域：

- ESP32-C3-MINI-1。
- LDO、电阻、电容。
- 调试焊盘。

也可以把 ESP32-C3-MINI-1 放正面上方，但必须保证天线区域不被 LED 金属件、USB 外壳或外壳金属遮挡。

## 天线约束

- ESP32-C3-MINI-1 天线端必须靠 PCB 边缘。
- 天线下方和前方按 Espressif datasheet 做 keepout。
- keepout 区域不要铺铜、不要走线、不要放螺丝、不要靠金属外壳。
- 如果外壳有金属件，金属件必须远离天线端。

## USB

- USB-C 靠边放置。
- D+ / D- 短且并行，尽量等长。
- CC1/CC2 各接 5.1k 到 GND。
- USB 口外壳接 GND，按工程师习惯处理机壳地/板地。

## 电源

- 5V 主线从 USB-C 到 LDO、板载 WS2812B 供电路径、WS2812 预留焊盘。
- ESP32-C3 的 3V3 旁必须有 10uF + 100nF 去耦。
- LDO 输入输出电容紧贴 LDO。
- 板载 D1 使用 D3 形成 `LED5V`，保证 3.3V 数据高电平余量。
- 若未来真接 WS2812 灯环，5V/GND 走线要加宽，并安装 C6 470uF。

## 板载 WS2812B

- 放 PCB 正中心，方便扩散罩均匀出光。
- GPIO4 经 R4 330 ohm 串到 D1.DIN。
- D1.DOUT 接扩展焊盘 P1 DATA。
- D1 旁放 100nF 去耦。
- 丝印标出 D1 方向，避免贴反。

## 按钮

- SW1 放边缘，用户可通过外壳按到。
- 接 GPIO10 到 GND。
- 不使用 C3 的 BOOT/IO9 做产品清网按钮。

## 扩展焊盘

- WS2812 焊盘放背面或边缘，丝印 `5V DIN GND`。
- 如果要做到完全用户无焊接，V1 可以不对外暴露这些焊盘，只作为工程预留。

## 交付给打板/贴片方

最终应输出：

- KiCad 或立创EDA 工程文件
- Gerber
- Drill
- BOM
- CPL/坐标文件
- 装配图
- 本目录的设计说明

本文件和 `v1_schematic.md` 是画板输入，不是生产文件。
