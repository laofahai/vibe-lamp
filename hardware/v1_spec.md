# Vibe Lamp Core V1 硬件规格

## 目标

V1 是一块小型桌面状态灯主控板，目标是“工厂贴好，用户不焊接”。用户拿到后只需要插 USB-C 供电，完成 WiFi 配网即可使用。

本版使用板载单颗 WS2812B 数字 RGB LED 作为主显示，预留 WS2812 灯环/灯带扩展焊盘。D1 是链上第一颗，外接扩展从 D1.DOUT 继续级联。

## 外形

- PCB：圆角方形，当前 Core V1 为 40 mm x 40 mm。
- 层数：2 层板即可。
- 厚度：1.0 mm 或 1.2 mm，便于做小外壳。
- 正面中心放 WS2812B 状态灯，外壳使用半透明扩散罩。
- USB-C 口放一边，作为供电和调试接口。
- 清网按钮放侧边或背面边缘，避免误触。

## 主控

- MCU：ESP32-C3-MINI-1 模组。
- 原因：
  - 体积比 ESP32-C3 SuperMini 载板小。
  - 自带射频匹配和天线，降低 RF 风险。
  - 支持原生 USB，后续刷机/日志更方便。
- 组装方式：SMT 工厂贴片，不要求用户焊接。

## 显示

### 主显示：板载 WS2812B

- 类型：WS2812B 数字 RGB LED。
- 立创料号：C2761795。
- DATA：GPIO4，经 R4 330 ohm 串到 D1.DIN。
- D1.DOUT 接扩展焊盘 P1 的 DATA。
- D1 就近放 C7 100 nF 去耦。
- D3 使用 1N4148WS 串在 LED 供电路径形成 `LED5V`，把 D1 供电从 USB 5V 降到约 4.3V，降低 WS2812B 输入高电平门槛，保证 ESP32-C3 的 3.3V GPIO 有足够余量。

### 预留扩展：WS2812

- 预留 3 个测试焊盘或小焊盘：
- 5V
- DATA
- GND
- DATA 来自 D1.DOUT。
- 外接灯带/灯环按总灯数重新配置固件 `NUM_LEDS`。
- V1 默认不装外接灯环，不要求用户焊接；这是给 V2 或工程验证留的扩展。

## 按钮

- 清网按钮：GPIO10 到 GND，输入上拉。
- 功能：长按清除 WiFi 凭据并重启进入配网。
- 按钮必须工厂贴好，用户不焊接。

## 供电

- 输入：USB-C 5V。
- 3.3V：板上 LDO 给 ESP32-C3-MINI-1 供电。
- 单颗 RGB 供电电流很小，普通 USB 口足够。
- WS2812 只是预留，若未来外接灯环，要按灯数重新核算 5V 电流。

## 固件目标

V1 主固件使用 `c3_core_v1` env：

- `DISPLAY_TYPE=DISPLAY_WS2812_STRIP`
- `NUM_LEDS=1`
- `PIN_WS2812=4`
- `PIN_RESET_BTN=10`

外接 WS2812 灯环/灯带时仍可复用 `c3_core_v1`，按总灯数调整 `NUM_LEDS`。

## 打样/生产要求

- 必须选择 SMT 贴片服务。
- USB-C、ESP32-C3-MINI-1、LDO、RGB LED、按钮、电阻电容都应由工厂贴好。
- 用户不做任何焊接。
- 如果找硬件工程师画板，把本目录下的 `v1_schematic.md`、`v1_bom.csv`、`v1_pcb_constraints.md` 一起给对方。
